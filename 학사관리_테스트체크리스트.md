## STEP 2. 사전 준비
- [ ] 교사/관리자 계정으로 로그인 → 사이드바에 **🏫 학사** 메뉴가 보이는지
- [ ] (선택) 관리자 → 학생 그룹(반)을 1~2개 만들고 학생 몇 명 배정
- [ ] (증명서용) 관리자 → 공동체 설정 → **기록부 기관** 등록(로고 포함)

---

## STEP 3. 모듈별 테스트

### 📋 출석부
- [ ] 학사 → 출석부, 오늘 날짜 학생 명단이 모두 '출석'으로 표시
- [ ] 몇 명을 지각/결석으로 바꾸고 **저장** → 새로고침 후에도 유지되는지
- [ ] 날짜 ‹ › 이동, 반 필터 동작
- [ ] **월간 통계** 탭 → 학생별 수업일수/출석률 표시
- [ ] **TCS 반영** → 해당 학생 TCS 기록부 출결에 숫자가 들어가는지

### 🗓 시간표
- [ ] 반(또는 공통) 선택 → 칸에 과목·교사·교실 입력 후 **저장** → 유지되는지
- [ ] **교사별 보기** → 특정 교사의 수업이 여러 반에서 모여 보이는지
- [ ] **🖨 인쇄** → A4 가로 시간표 출력 미리보기

### 💰 학비
- [ ] 일괄 청구: 반/전체 · 항목 · 금액 · 기한 입력 → **청구 생성**
- [ ] 학생별 **미납→납부완료** 토글, 요약(청구/수납/미납) 숫자 확인
- [ ] 전체/미납/완료 · 청구월 · 반 필터 동작

### ✉️ 가정통신문
- [ ] 새 통신문 작성(대상 전체/반) → 저장
- [ ] 학생/멤버 계정으로 로그인 → **홈 화면**에 통신문 카드가 보이는지
- [ ] **📄 PDF/인쇄** → ';PDF로 저장' 으로 파일 생성 → 학부모에게 전달용
- [ ] **🔗 공유** → 링크 복사 → 로그아웃 상태(또는 시크릿창)에서 링크 열어 열람되는지
- [ ] **공유 해제** 후 같은 링크가 막히는지

### 📄 증명서
- [ ] 학생 선택 → 성명·생년월일·학년·학교 자동 채움
- [ ] 종류(재학/졸업/직접입력) 선택 → **미리보기 & 인쇄** → 공문서 형식 확인
- [ ] 발급 기관 로고·직인 표기 확인

### 💬 상담기록 / 🏥 보건
- [ ] 학생 선택 → 분류·날짜·제목·내용 입력 → **＋ 기록 추가**
- [ ] 날짜순으로 쌓이고 분류 색상 배지 표시, 삭제 동작
