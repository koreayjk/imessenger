# 학사관리(학교 행정) 기능 — 설치 안내

작은 학교가 성적·종합기록부 외에 **학교 행정 전반**을 이 앱 하나에서 처리할 수 있도록
사이드바에 **🏫 학사** 메뉴를 추가했습니다.

## 들어가는 모듈

탭은 **학생 중심**으로 재구성됐습니다. (타 학교시스템(SIS)처럼 학생 1명 = 한 화면)

| 탭 | 기능 | 상태 |
|---|---|---|
| 👤 학생 | **학생 1명의 모든 것을 한 페이지에** — 담당교사 배정, 이번달 출결·미납 요약, 상담·생활지도, 보건, 종합기록부 출력 | ✅ 완료 |
| 📋 출석부 | 일일 전자출석부 — 반별 출석체크 → TCS 출결 자동 반영 | ✅ 완료 |
| 🗓 시간표 | 반별·공통 주간 시간표 (과목·교사·교실) | ✅ 완료 |
| 💰 학비 | 일괄 청구·납부 체크·미납 현황 | ✅ 완료 |
| ✉️ 가정통신문 | 통신문 작성 → 가족이 홈에서 열람 / PDF 전달 | ✅ 완료 |
| 📄 증명서 | 재학·졸업·직접입력 증명서 발급/인쇄 (DB 불필요) | ✅ 완료 |

> **상담·보건은 별도 탭이 아니라 👤 학생 페이지 안**으로 들어갔습니다.
> 학생을 누르면 그 학생의 출결·학비·상담·보건·종합기록부가 **한 화면**에 모두 나옵니다.

### 👤 학생 통합 뷰 사용법
1. 사이드바 **🏫 학사 → 👤 학생** (기본 첫 화면)
2. 왼쪽 목록에서 학생 선택 — **전체 / 내 담당** 필터, 반 필터 가능
3. 상단에서 **담당교사** 지정 (드롭다운) → "내 담당"으로 본인 학생만 모아 보기
4. 한 페이지에서: 이번달 출결·미납 요약 / 상담·생활지도 추가 / 보건 기록 추가
5. **종합기록부**:
   - **✏️ 기록부 편집** → 그 학생의 TCS 기록부(인적·성적·발달·진로 등) 편집 화면으로 (← 학생 버튼으로 복귀)
   - **📄 미리보기 / 출력** → 상단 **‘포함 항목’ 체크박스로 들어갈/뺄 항목 선택** 후 PDF 저장

> **구조 통합**: 학생 선택은 **👤 학생 한 곳**으로 일원화했습니다.
> 교사용 별도 'TCS' 사이드 메뉴는 없어지고, 종합기록부는 학생 페이지에서 편집·출력합니다.
> (학사관리 = 일상 운영 / TCS 종합기록부 = 그 결과로 나오는 공식 문서)

### 담당교사 기능 DB 설치 (한 번만)
`학사_담당교사_DB설치.sql` 을 Supabase SQL Editor 에 실행하세요 (members에 컬럼 1개 추가).
> 설치 전에는 담당교사 지정 시 안내 토스트가 뜹니다. 나머지 학생 기능은 설치 없이도 작동합니다.

## 1. 출석부 DB 설치 (한 번만)

`학사_출석부_DB설치.sql` 내용을 **Supabase → SQL Editor** 에 붙여넣고 **Run** 하세요.
`attendance_daily` 테이블과 인덱스, RLS 정책이 생성됩니다.

> 설치 전에는 학사 → 출석부에 "데이터베이스 설정이 필요해요" 안내가 표시됩니다.

---

## 2. 사용법 — 일일 전자출석부

1. 사이드바 **🏫 학사 → 출석부** (교사·관리자만 보임)
2. 상단에서 **날짜**와 **반(학생 그룹)** 선택 — 기본값은 오늘, 전체 반
3. 모든 학생은 처음에 **출석**으로 표시됩니다. 결석·지각·조퇴·병결·공결인 학생만 눌러서 바꾸세요
   - **전체 출석** 버튼으로 한 번에 초기화 가능
4. **저장** → 그 날짜의 출결이 저장됩니다 (다시 열면 그대로 불러옴)
5. **월간 통계** 탭에서 학생별 누적(수업일수·결석·지각·조퇴·출석률)을 확인하고,
   **TCS 반영** 버튼으로 해당 학생의 종합기록부 출결 섹션에 자동 기록할 수 있어요

### 출결 상태
- **출석 / 지각 / 조퇴 / 결석 / 병결 / 공결** 6가지
- TCS 출결로 반영할 때: 지각→지각, 조퇴→조퇴, 결석→결석 으로 집계됩니다
  (병결·공결은 통계에는 보이지만 종합기록부 '결석' 수에는 더해지지 않습니다)

---

## 3. 시간표 DB 설치 (한 번만)

`학사_시간표_DB설치.sql` 내용을 **Supabase → SQL Editor** 에 붙여넣고 **Run** 하세요.

### 사용법 — 주간 시간표
1. 사이드바 **🏫 학사 → 시간표**
2. 상단에서 **반**을 선택 (또는 **공통(전체)** — 모든 반 공통 시간표)
3. 교시 수를 선택하고, 각 칸에 **과목·교사·교실**을 입력
4. **저장** → 선택한 반의 시간표가 저장됩니다 (다시 열면 그대로 불러옴)
5. **교사별 보기**: 상단 토글로 전환하면 한 교사의 주간 수업을 모든 반에서 모아 봅니다
6. **🖨 인쇄**: 반별 시간표 / 교사별 시간표를 A4(가로) 형식으로 출력

---

## 4. 학비 DB 설치 (한 번만)

`학사_학비_DB설치.sql` 내용을 **Supabase → SQL Editor** 에 붙여넣고 **Run** 하세요.

### 사용법 — 학비/납부 관리
1. 사이드바 **🏫 학사 → 학비**
2. **＋ 일괄 청구 만들기**: 반(또는 전체) · 항목명 · 청구월 · 금액 · 납부기한을 입력하고
   **청구 생성** → 대상 학생 전원에게 청구가 한 번에 만들어집니다
3. 목록에서 학생별 **미납/납부완료** 버튼을 눌러 납부 처리 (납부일·금액 자동 기록)
4. 상단 요약 카드로 **청구 총액 · 수납 완료 · 미납액**을 한눈에 확인
5. 필터: 전체/미납/완료 · 청구월 · 반 별로 좁혀 볼 수 있어요 (기한 지난 미납은 ⚠ 표시)

## 5. 가정통신문 DB 설치 (한 번만)

`학사_가정통신문_DB설치.sql` 내용을 **Supabase → SQL Editor** 에 붙여넣고 **Run** 하세요.

### 사용법 — 가정통신문
1. (교사) 사이드바 **🏫 학사 → 가정통신문 → ＋ 새 가정통신문**
2. 제목 · 대상(전체/특정 반) · 발송일 · 내용을 입력하고 **저장**
3. **가족(학생·학부모)** 은 로그인 후 **홈 화면**의 ‘✉️ 가정통신문’ 카드에서 바로 읽습니다
   - 대상을 ‘전체’로 하면 모든 가족에게, 특정 반으로 하면 그 반 가족에게만 보입니다
4. 각 통신문은 **📄 PDF/인쇄** 버튼으로 출력할 수 있어요
   - 인쇄 대화상자에서 **‘PDF로 저장’** 을 고르면 PDF 파일이 됩니다 → 문자·카톡·메일로 학부모에게 전달
   - 또는 그대로 종이로 인쇄해 배부

### (선택) 학부모에게 링크로 전달 — 공유 링크 (로그인 불필요)
- 추가 설치: `학사_가정통신문_공유_DB설치.sql` 를 **Supabase → SQL Editor** 에서 실행
  (newsletters 테이블을 만든 뒤에 실행하세요)
- 통신문의 **🔗 공유** 버튼을 누르면 링크가 복사됩니다 → 문자/카톡으로 학부모에게 전달
- 학부모는 **로그인 없이** 링크만 열면 통신문을 읽고 인쇄할 수 있어요
- **공유 해제** 를 누르면 그 링크는 즉시 막힙니다
- 보안: 교사가 **명시적으로 공유한 통신문만** 비로그인 열람이 가능합니다 (나머지는 계정 필요)

> **학부모 계정 연동(예정)**: 현재는 학생 계정으로 홈에서 통신문을 봅니다.
> 학부모 전용 계정·자녀 연동·자녀별 출결/학비 열람은 다음 단계로 추가할 수 있어요.

## 6. 증명서 발급 (DB 설치 불필요)

학생·학적 정보와 발급 기관(로고 포함)을 재사용하므로 **별도 테이블이 필요 없습니다.**

### 사용법 — 증명서
1. 사이드바 **🏫 학사 → 증명서**
2. **학생**을 선택하면 성명·생년월일·학년·학교가 자동으로 채워집니다 (필요시 수정)
3. **증명서 종류**: 재학증명서 / 졸업증명서 / 직접입력
4. 발급 사유·발급일·발급번호·발급 기관·대표자 명의를 확인하고 **미리보기 & 인쇄**
5. A4 공문서 형식으로 출력됩니다 (기관 로고·직인 표기 포함)

> 발급 기관(로고)은 **관리자 → 공동체 설정 → 기록부 기관**에서 등록합니다.

## 7. 상담·생활지도 / 보건 DB 설치 (각각 한 번만)

- 상담: `학사_상담_DB설치.sql`
- 보건: `학사_보건_DB설치.sql`

각 파일 내용을 **Supabase → SQL Editor** 에 붙여넣고 **Run** 하세요.

### 사용법 — 상담기록 / 보건
1. 사이드바 **🏫 학사 → 상담기록** 또는 **보건**
2. **학생**을 선택
3. 분류(상담/생활지도/칭찬/주의 · 보건실방문/투약/병결/예방접종 등) · 날짜 · 제목 · 내용 입력
4. **＋ 기록 추가** → 학생별로 날짜순 누적됩니다 (분류별 색상 배지로 한눈에 구분)
