# 모의고사 AI 분석 기능 — 설치 안내

앱(index.html)은 이미 준비됐습니다. Gemini 키를 안전하게 숨기기 위해 **Supabase Edge Function** 하나만 배포하면 작동합니다.

---

## 1. 폴더 구조 만들기

프로젝트 루트에 아래처럼 폴더를 만들고, `analyze-mockexam.index.ts` 파일의 내용을
`index.ts` 로 넣습니다.

```
supabase/
  functions/
    analyze-mockexam/
      index.ts   ← analyze-mockexam.index.ts 내용을 그대로 복사
```

---

## 2. Supabase CLI 준비 (한 번만)

```bash
npm install -g supabase        # 또는: brew install supabase/tap/supabase
supabase login                 # 브라우저로 로그인
supabase link --project-ref bbfvnmlrdtzmcwfadjqw
```

> project-ref 는 앱의 SUPABASE_URL(bbfvnmlrdtzmcwfadjqw) 에서 가져온 값입니다.

---

## 3. Gemini 키를 시크릿으로 등록

```bash
supabase secrets set GEMINI_API_KEY=여기에_발급받은_Gemini_키
```

키는 Google AI Studio(aistudio.google.com)에서 무료로 발급됩니다.
**키는 절대 index.html 이나 코드에 직접 넣지 마세요.** 위 명령으로만 등록합니다.

---

## 4. 함수 배포

```bash
supabase functions deploy analyze-mockexam --no-verify-jwt
```

- `--no-verify-jwt` 는 앱에서 바로 호출되도록 하기 위한 옵션입니다.
- 앱의 함수 이름(`analyze-mockexam`)과 폴더 이름이 같아야 합니다.

---

## 5. 사용법

1. 앱 → TCS 탭 → 학생 선택 → **🧠 모의고사 분석** 카드 클릭
2. 시험지 스캔(사진·PDF) 업로드 — 정답지나 점수표가 있으면 함께 올리면 정확도 ↑
3. (선택) 시험명·과목·정답 입력
4. **AI 분석하기** → 과목·약점·집중 학습·학습법이 표시됨
5. **기록부에 저장** → 학생별로 누적 저장 (PDF 출력에도 포함됨)

---

## 참고 / 주의

- **정확도**: 정답·점수가 함께 있으면 정밀 채점, 없으면 AI가 직접 풀어 추정 채점(참고용)합니다.
  수학·과학은 추정도 잘 맞지만, 국어·암기형은 불확실할 수 있어 결과에 "참고용"으로 표시됩니다.
- **모델 변경**: 더 똑똑한 분석이 필요하면 `index.ts` 의 `MODEL` 을 `"gemini-3.5-flash"` 로 바꾸세요.
  (gemini-3.x 는 temperature 기본값을 권장하므로, 바꿀 경우 `generationConfig` 에서 `temperature: 0.2` 줄을 지우는 걸 권장합니다.)
- **보안 강화(선택)**: `--no-verify-jwt` 로 배포하면 함수 URL을 아는 사람은 호출할 수 있습니다.
  남용이 걱정되면 함수에 공유 비밀 헤더 검사나 호출 횟수 제한을 추가할 수 있어요. 필요하면 말씀해 주세요.
- **비용**: Gemini 무료 등급으로 시작 가능하며, 시험지 이미지 분석은 호출당 매우 저렴합니다.
